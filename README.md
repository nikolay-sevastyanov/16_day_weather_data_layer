# Data-pipeline погоды в Алматы
## Инструменты: Apache Airflow, Docker, Python, PostgreSQL

Слой данных с погодой в Алматы позволит компаниям, имеющим филиалы в этом городе сможет помочь планировать внутренние процессы и коммерческую активность, в зависимости от связи погоды и иных метрик.

Шаги подготовки data-pipeline:

1. Создание и активация виртуального окружения py_env для корректной работы docker
1. Установка Docker для Apache Airflow 3.1.3
2. Установка базы данных PostgreSQL 18 с заготовленной пустой базой данных "weather".
3. Написание кода DAG на языке Python для автоматический подгрузки как и реальной температуры (ближайший час), так и прогнозной (каждый час 16 дней вперед) для сравнения погрешности.
4. Развертывание Airflow в Docker
5. Запуск БД PostgreSQL
6. (опционально, индивидуально) подлючение Power BI к PostgreSQL для выведения дэшбордов погоды.

Результат: дополнительный, вспомогательный слой данных, который позволит планировать закупку, найм персонала, нагрузку серверов и других процессов в зависимости от предной области и специфики бизнеса.
